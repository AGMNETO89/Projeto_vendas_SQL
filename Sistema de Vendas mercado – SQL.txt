 Sistema de Vendas/ mercado – SQL

 Descrição
Projeto de banco de dados relacional desenvolvido em SQL, simulando um sistema real de vendas.
O projeto contempla modelagem de dados, integridade referencial, transações, consultas com JOIN e relatórios analíticos.

 Objetivo
Demonstrar domínio prático de SQL aplicado ao dia a dia corporativo, incluindo:
- Criação e relacionamento de tabelas
- Manipulação de dados
- Consultas complexas
- Controle de transações

 Estrutura do Banco
- clientes
- produtos
- pedidos
- itens_pedido

Relacionamentos:
- Um cliente pode ter vários pedidos
- Um pedido pode ter vários itens
- Um item pertence a um produto

 Tecnologias
- MySQL
- SQL padrão
- Modelagem relacional
 Funcionalidades
- Cadastro de clientes e produtos
- Criação de pedidos
- Inclusão de itens no pedido
- Cálculo automático de total
- Transações seguras (COMMIT / ROLLBACK)
- Relatórios com JOIN, ORDER BY e subqueries

 Consultas Implementadas
- Último pedido realizado
- Pedidos por cliente
- Faturamento por cliente
- Ranking de produtos mais vendidos
- Pedidos sem itens (LEFT JOIN)

  Aprendizados
- Normalização de dados
- Integridade referencial
- SQL aplicado a sistemas reais
- Pensamento analítico e relacional
